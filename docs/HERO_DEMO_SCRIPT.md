## üé¨ **Recording Script - "AI Research Agent"**

**Duration:** 25-30 seconds (will speed up to 20s in post-production)
**File name:** `hero-research-agent.mp4`

### **The Story:**
"Watch an AI agent perform comprehensive web research and generate a professional report in seconds"

---

### **Action-by-Action Script:**

```
[0:00] üé¨ START RECORDING (Press F9)
       - Baby Research workflow is loaded and visible
       - All 8-10 nodes are visible on canvas
       - Cursor is off to the side

[0:02] - Zoom out slightly (Ctrl/Cmd -) if needed
       - Make sure all nodes are visible in frame
       - Workflow should look impressive and complex

[0:04] - Move cursor to "Run Workflow" button (top right)
       - Hover over it briefly

[0:05] - Click "Run Workflow" button
       - Input modal appears

[0:06] - PAUSE (0.5 seconds)
       - Let viewers see the input modal

[0:07] - Type in the input field:
       - "Research the top 3 AI coding assistants in 2025"
       - OR: "What are the best AI tools for developers?"
       - Type at normal speed (not too fast)

[0:11] - Click "Run" or "Execute" button
       - Modal closes
       - Workflow starts executing

[0:12] - Watch nodes light up in sequence:
       - INPUT node (green/blue glow)
       - INITIAL_SEARCH node (executing)
       - ANALYSIS node (executing)
       - CONTENT_EXTRACTION node (executing)
       - SUMMARY node (executing)
       - OUTPUT node (final result)

[0:22] - Nodes finish executing
       - Show the final OUTPUT node with results
       - Maybe click on OUTPUT to show the report preview

[0:25] - PAUSE (2-3 seconds)
       - Let viewers see the final result
       - Show the research report if visible

[0:28] üõë STOP RECORDING (Press F10)
```

**Note:** The actual execution might take 10-20 seconds. That's OK! We'll speed it up to 2x in post-production to make it 10-12 seconds total.
